エクスプローラー風の2ページ見開き表示可能な画像ビューワー。まだまだ開発中。
サムネイル一覧はマルチスレッドで画像読み込みを行うので、SSDならそこそこ高速に読み込みます。

基本機能は、けんじ氏作成のフリーソフト [Leeyes](http://www3.tokai.or.jp/boxes/leeyes/) を参考にしていますが、
自分の欲しい機能のみ実装するつもりで、フル機能のカバーまでは目指しません。

## 対応画像形式
jpeg, png, gif(アニメーション不可), bmp, tif, webp, dds

## 基本操作
メインウィンドウ左側にフォルダツリーが表示され、フォルダをクリックすると右側にサムネイル一覧が表示されます。

- 大きい画像ウィンドウ表示

サムネイルをダブルクリック

- 画像ウィンドウ次ページ

マウスホイール下回転 または キーボード→キー

- 画像ウィンドウ前ページ

マウスホイール上回転 または キーボード←キー

## インストール・アンインストール
インストーラーはないので、適当な場所にZIP解凍して実行してください。
アンインストールはEXEのフォルダと設定ファイルを手動で消してください。
設定ファイルは "ドキュメント/RadianTools/IMV/IMV.config.json" に保存されます。
